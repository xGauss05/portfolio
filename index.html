Alien: Nemesis

In Project III, the whole class of 23 students worked together in doing this videogame. In this project I was helping in developing some of the modules required for making the game, such as the scripting engine and their Internal Calls, and the very beginning of the lighting module.

By my hand, I created TheOneBuilder that acted as a new entry point for the whole .sln project, which allows the project to generate the .exe by getting all the .dll files, the scenes and the assets in a separate folder.

I also created the logic for the ItemManager. It comprises of 3 different dictionaries, 1 for the item data, 1 for the inventory and 1 for the equipped items.

And last but not least, ALL the behaviors from the enemies, including the first version of the Alien Queen.

Features done in our own Game Engine created from scratch, TheOneEngine. All the assets were done from scratch, models, sounds, textures.

Check it out at Alien: Nemesis Official Page.
